# react-redux-router-boilerplate

All-in boilerplate for implementing react-redux-router projects

```
Start:
> git clone https://github.com/nikorudnas/react-redux-router-boilerplate.git
> cd react-redux-router-boilerplate
> yarn install
> yarn start
```

```
Build:
> yarn build
```

```
Test:
> yarn test
```
